NetTrack Pro V13.8 Subscription Signal Update

Applied:
- Signal A+B button remains the only active signal generator button.
- Free Tester / Basic: 2 signals per day, TP1 only.
- Pro: 4 signals per day, TP1 + TP2.
- Premium / Elite / Owner: 8 signals per day, TP1 + TP2 + TP3.
- Every generated signal now shows Entry, Stop Loss, TP1, and locked/unlocked TP2/TP3 based on subscription.
- Risk:Reward display added.
- Copy Signal helper added in code for future button wiring if needed.

No major layout redesign was applied.
