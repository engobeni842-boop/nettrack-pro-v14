NetTrack Pro Web Deploy Package

1. Upload this folder or the ZIP to Netlify.
2. The main file is index.html.
3. For sign-up/login/API verification, add Supabase or Firebase Auth next.
4. Do not expose private API keys in index.html. Use serverless functions for secret keys.
